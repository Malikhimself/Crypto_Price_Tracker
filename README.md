Crypto Price Tracker
- A real-time cryptocurrency price tracking application that monitors digital asset prices and provides up-to-date market information.

Features:
- Real-time price updates for major cryptocurrencies
- Clean and intuitive user interface
- Price change indicators and percentage movements
- Historical price data visualization
- Support for multiple crypto assets

Technologies used:
- Next.js
- TypeScript
- Tailwind CSS
- Chart.js

APIs:
This project uses the following APIs for fetching cryptocurrency data:
- CoinGecko API
- TradingView API
- Binance API

## Getting Started

# Clone the repository
git clone https://github.com/Malikhimself/Crypto_Price_Tracker.git

# Navigate to project directory
cd Crypto_Price_Tracker

# Install dependencies
npm install

# Start the application
npm start

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
